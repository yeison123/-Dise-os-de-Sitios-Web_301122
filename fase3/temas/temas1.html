Curso de HTML

Los objetivos del curso de HTML online, son:

  • Conocer cuál es la base de la programación de páginas Web.

    • Etiquetas HTML básicas

    Existen un grupo de etiquetas que a mi parecers, correspondenta a las etiquetas basicas para el modelado de una pagina Web y son las más usadas en HTML, este articulo contiene algunas de ellas:

    • <div> Son contenedores utilizados para agrupara elementos en bloques
    • <a> Utilziada para agregar enlaces
    • <body> utilizada en el modelado de contenidos
    • <H1>…<H6> Utilizadas en la aplicacion de titulos
    • <img> permite agregar imagenes a un documentos HTML
    • <head> Permite la descripcion del documento
    • <strong> Permite agregar negrilla a los textos
    • <p> Permite ordenar los textos en parágrafos
    • <br> Agrega saltos del linea
    • <ol> Perite agergar listas ordenasda, como la que vemos en este mometo estas se se inclyen dentor de <ul> estas a su ves permiten listar, <li>

    Adicional a estos elementos existe otos que le dan la estructura base a un archivo HTML

    <body> </body>

    Este elemento establece el cuerpo del documentos y es fundamental para que los motores debusqueda establescas los contenidos visibles del archivo HTML.

    <head></head>

    Esta parte del archivo en fundamental para establecer elementos de relevancia para los buscadores a la hora de posicionar y clasificar la pagiana en esta parte del archivo se almacenan datos como:

    • Título
    • Hojas de estilos
    • JavaScript
    • CSS, etc
