Propósito del sitio

El objetivo del Aula Virtual Virtual School es proveer un espacio educativo en el que tanto los docentes como los alumnos puedan desarrollar sus actividades académicas y de investigación, encontrando en estos entornos una comunidad educativo de acceso publico y gratuito, en donde puedan intercomunicarse mediante el uso de herramientas que soporten y faciliten sus procesos de enseñanza-aprendizaje a través de la WEB.

Plataformas educativas y formación online

Curso HTML

Los objetivos del curso de HTML online, son:

  • Conocer cuál es la base de la programación de páginas Web.
  • Dominar el lenguaje de marcación HTML.
  • Etiquetas y atributos. Utilizar etiquetas para definir los formatos de las páginas Web, los textos que las componen, imágenes, listas y esquemas, hipervínculos, marcadores, tablas, páginas de marcos, formularios y elementos decorativos.
  • Incluir hojas de estilo en un página Web.
  • Aprender la definición de clases y capas. CSS.

Curso Angular

El objetivo de este curso es introducir Angular 5, para desarrolladores con pocos o ningún conocimiento de este framework Javascript. Veremos a la largo de varias sesiones las herramientas principales que se disponen en Angular para desarrollar, junto con las piezas fundamentales que los programadores deben crear en el marco de una aplicación.
